结论分析与体会：

​ 本次使用需要在 github 上进行多次操作，还有一些关于 hexo 的知识，其中也包含了很
多问题，比如 github 在不使用加速器的情况下难以进入页面，或者使用了加速器但是出于
未知原因，使用仓库出现 404 的原因，还有链接 github 仓库会出现端口拒绝链接的问题。
我通过在网上搜寻解决这些问题的办法，从而解决这些问题并对整个实验有更多的了解。
通过注册 Github 账号和搭建 Hexo 环境，可以实现个人博客的搭建。Github 是一个全
球最大的开源社区，拥有大量的开发者和技术爱好者，可以为个人博客提供丰富的技术支持
和社交平台。而 Hexo 是一个快速、简洁且高效的博客框架，可以帮助用户快速搭建个人博
客。
​ 在使用 Hexo 搭建个人博客时，需要掌握一定的技术知识，如 Markdown 语法、Git 版本
控制等。同时需要选择一个适合自己的主题，以及进行一定的定制化配置，如域名解析、SEO
优化等。通过这些工作，可以让个人博客更具个性化和专业性。总之，搭建个人博客需要不
断学习和努力，但是通过这个过程可以获得很多有价值的东西，是值得尝试和坚持的。
